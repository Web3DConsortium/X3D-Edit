X3D-Edit is an Extensible 3D (X3D) Graphics authoring tool for simple error-free creation, editing, validation and viewing of X3D scenes for interactive Web-based visualization. Extensible 3D (X3D) Graphics is the royalty-free open standard for publishing, viewing, printing and archiving interactive 3D models on the Web. X3D-Edit runs as a standalone application or NetBeans plugin. The X3D file format is an advanced XML-based version of the original VRML97 international standard for 3D scene graphs.
