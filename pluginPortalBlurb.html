X3D-Edit is an Extensible 3D (X3D) Graphics authoring tool for simple error-free creation, editing, validation and viewing of X3D models for interactive Web-based visualization. X3D-Edit runs as a standalone application or NetBeans plugin. X3D Graphics is the royalty-free open International Standard for publishing, viewing, printing and archiving interactive 3D models on the Web. The X3D file format is an advanced XML-based version of the original VRML97 international standard for 3D scene graphs. The non-profit Web3D Consortium maintains this codebase and the X3D family of standards.
